Uploaded Image

When you upload an image, it goes into our computer vision processor that searches for a sudoku grid. Then, the image is reshaped into a proper square grid, and each of the 81 number's images is extracted and passed on to the next step  

Extracted Sudoku

All the 81 extracted digits images are individually passed to an MNIST trained Deep Learning model to get the numerical value. This matrix is then passed to the last solver

Final Solution

There are 6,670,903,752,021,072,936,960 9x9 valid sudoku grids. So, brute force is not feasible. Hence, we use the backtracking approach to solve it. Even though solving sudoku is an NP-complete problem, we can solve it in moments because of the fixed 9x9 size
